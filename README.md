# 🧤 Sign Language to Speech Gloves

## 📌 Overview

Communication is the transmission of information through speech, text, images, and other mediums. In today’s data-driven society, effective communication is essential for everyday life.

However, individuals who are deaf or hard of hearing often face challenges in verbal communication. To address this, this project presents a **Sign Language to Speech Glove system** that bridges the communication gap between sign language users and the hearing community.

The system focuses on **American Sign Language (ASL)**, a widely used natural sign language in North America.

---

## 🎯 Project Aim

The aim of this project is to develop an assistive wearable system that:

- Captures hand gestures used in ASL
- Classifies gestures into alphabets/words
- Converts recognized gestures into speech output

This enables seamless communication between sign language users and non-signers.

---

## 🧠 Key Idea

The glove uses sensor-based motion detection to interpret hand gestures:

- **5 Flex Sensors** → Detect finger bending and hand posture
- **MPU6050 Gyroscope** → Captures hand orientation and motion
- **Microcontroller** → Processes sensor data and runs classification logic
- **Speaker Module** → Converts predicted output into audible speech

---

## 🧩 System Workflow

1. User wears the glove
2. Performs ASL gesture (alphabet/word)
3. Sensors capture finger and hand movement
4. Data is processed and classified
5. Corresponding text output is generated
6. Speech module converts text to audio output

---

## 📷 Components Used

- Flex Sensors (5 units)
- MPU6050 Gyroscope

## 🔬 Applications

- Teaching and learning American Sign Language

## 🌍 Importance

This system helps break communication barriers by enabling real-time translation of sign language into speech, improving accessibility and inclusivity in daily interactions.
